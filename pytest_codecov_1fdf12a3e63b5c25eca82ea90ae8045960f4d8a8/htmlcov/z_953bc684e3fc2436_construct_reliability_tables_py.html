21 """Populate reliability tables for use in reliability calibration. 

22 

23 Loads historical forecasts and gridded truths that are compared to build 

24 reliability tables. Reliability tables are returned as a cube with a 

25 leading threshold dimension that matches that of the forecast probability 

26 cubes and the thresholded truth. 

27 

28 Args: 

29 cubes (list of iris.cube.Cube): 

30 A list of cubes containing the historical probability forecasts and 

31 corresponding truths used for calibration. These cubes must include 

32 the same diagnostic name in their names, and must both have 

33 equivalent threshold coordinates. The cubes will be distinguished 

34 using the user provided truth attribute. 

35 truth_attribute (str): 

36 An attribute and its value in the format of "attribute=value", 

37 which must be present on truth cubes. 

38 n_probability_bins (int): 

39 The total number of probability bins required in the reliability 

40 tables. If single value limits are turned on, these are included in 

41 this total. If using single_value_limits this value must be at 

42 least 3. 

43 single_value_lower_limit (bool): 

44 Mandates that the lowest bin should be single valued, with a small 

45 precision tolerance, defined as 1.0E-6. The bin is thus 0 to 1.0E-6. 

46 single_value_upper_limit (bool): 

47 Mandates that the highest bin should be single valued, with a small 

48 precision tolerance, defined as 1.0E-6. The bin is thus (1 - 1.0E-6) to 1. 

49 aggregate_coordinates (List[str]): 

50 An optional list of coordinates over which to aggregate the reliability 

51 calibration table using summation. This is equivalent to constructing 

52 then using aggregate-reliability-tables but with reduced memory 

53 usage due to avoiding large intermediate data. 

54 

55 Returns: 

56 iris.cube.Cube: 

57 Reliability tables for the forecast diagnostic with a leading 

58 threshold coordinate. 

59 """ 

60 from improver.calibration import split_forecasts_and_truth 

61 from improver.calibration.reliability_calibration import ( 

62 ConstructReliabilityCalibrationTables, 

63 ) 

64 

65 forecast, truth, _ = split_forecasts_and_truth(cubes, truth_attribute) 

66 

67 return ConstructReliabilityCalibrationTables( 

68 n_probability_bins=n_probability_bins, 

69 single_value_lower_limit=single_value_lower_limit, 

70 single_value_upper_limit=single_value_upper_limit, 

71 )(forecast, truth, aggregate_coordinates)
