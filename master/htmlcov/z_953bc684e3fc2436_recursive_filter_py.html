21 """Module to apply a recursive filter to neighbourhooded data. 

22 

23 Run a recursive filter to convert a square neighbourhood into a 

24 Gaussian-like kernel or smooth over short distances. The filter uses a 

25 smoothing_coefficient (between 0 and 1) to control what proportion of the 

26 probability is passed onto the next grid-square in the x and y directions. 

27 

28 Each iteration of the recursive filter applies the smoothing coefficients 

29 forwards and backwards in both the x and y directions. Applying 1-10 

30 iterations of the filter is typical. Each iteration further smooths the 

31 data, meaning the user must make a judgement regarding the number of 

32 iterations to apply that preserves real detail whilst removing artefacts 

33 in their data. 

34 

35 The IMPROVER plugin actually limits the maximum smoothing coefficient to 

36 a value of 0.5. Above this the smoothing is considered to be too great. 

37 The smoothing_coefficient can be set on a grid square by grid-square basis 

38 for the x and y directions separately (using two arrays of 

39 smoothing_coefficients of the same dimensionality as the domain). 

40 

41 Args: 

42 cube (iris.cube.Cube): 

43 Cube to be processed. 

44 smoothing_coefficients (iris.cube.CubeList): 

45 CubeList describing the smoothing_coefficients to be used in the x 

46 and y directions. 

47 iterations (int): 

48 Number of times to apply the filter. 

49 variable_mask (bool): 

50 Determines whether each spatial slice of the input cube can have a 

51 different mask. If False and cube is masked, a check will be made that 

52 the same mask is present on each spatial slice. If True, each spatial 

53 slice of cube may contain a different spatial mask. 

54 mask_zeros (bool): 

55 If set true all of the values of 0 in the cube will be masked, 

56 stopping the recursive filter from spreading values into these areas. 

57 They will then be unmasked later on. If the input cube was masked 

58 this mask will be reapplied to the output at the end. 

59 

60 Returns: 

61 iris.cube.Cube: 

62 The processed Cube. 

63 """ 

64 from improver.nbhood.recursive_filter import RecursiveFilter 

65 

66 plugin = RecursiveFilter(iterations=iterations) 

67 return plugin( 

68 cube, 

69 smoothing_coefficients=smoothing_coefficients, 

70 variable_mask=variable_mask, 

71 mask_zeros=mask_zeros, 

72 )
