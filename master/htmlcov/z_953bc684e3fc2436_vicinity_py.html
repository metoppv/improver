13def process( 

14 cube: cli.inputcube,

16 *, 

17 vicinity: cli.comma_separated_list = None, 

18 operator: str = "max",

21 """Module to apply vicinity processing to data. 

22 

23 Calculate the operator value within a vicinity radius about each point 

24 in each x-y slice of the input cube, with default being the maximum. 

25 

26 If working with thresholded data, using this CLI to calculate a 

27 neighbourhood maximum ensemble probability, the vicinity process must 

28 be applied prior to averaging across the ensemble, i.e. collapsing the 

29 realization coordinate. A typical chain might look like: 

30 

31 threshold --> vicinity --> realization collapse 

32 

33 Note that the threshold CLI can be used to perform this series of steps 

34 in a single call without intermediate output. 

35 

36 Users should ensure they do not inadvertently apply vicinity processing 

37 twice, once within the threshold CLI and then again using this CLI. 

38 

39 As a further note, when applying the mean operator to thresholded data, 

40 the vicinity CLI will produce output equivalent to the nhbood CLI 

41 (disregarding the boundary), but less efficiently (particularly when 

42 applied to masked or datasets containing NaNs). For such cases, it is 

43 recommended that one use the nbhood CLI instead. 

44 

45 Args: 

46 cube (iris.cube.Cube): 

47 A cube containing data to which a vicinity is to be applied. 

48 land_mask (iris.cube.Cube): 

49 Binary land-sea mask data. True for land-points, False for sea. 

50 Restricts in-vicinity processing to only include points of a 

51 like mask value. 

52 vicinity (list of float / int): 

53 List of distances in metres used to define the vicinities within 

54 which to search for an occurrence. Each vicinity provided will 

55 lead to a different gridded field. 

56 operator (str): 

57 Operation to apply over vicinity. Options are one of: ["max", "mean", "min", "std"] 

58 with "max" being the default. 

59 new_name (str): 

60 Name to assign to the resultant cube after calculating the vicinity 

61 values for the specified operator. Where no value is provided, the 

62 cube will retain the same name as the input cube. 

63 Returns: 

64 iris.cube.Cube: 

65 Cube with the vicinity processed data. 

66 """ 

67 from improver.utilities.spatial import OccurrenceWithinVicinity 

68 

69 return OccurrenceWithinVicinity( 

70 radii=vicinity, land_mask_cube=land_mask, operator=operator 

71 ).process(cube, new_name=new_name)
