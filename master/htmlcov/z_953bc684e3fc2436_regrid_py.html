32 land_sea_mask (iris.cube.Cube): 

33 Cube describing the land_binary_mask on the source grid if land-sea

37 regrid_mode (str): 

38 Selects which regridding techniques to use. Default uses 

39 iris.analysis.Linear(); "nearest" uses iris.analysis.Nearest(); 

40 "nearest-with-mask" uses Nearest() with land-sea awareness.

43 "bilinear-with-mask": bilinear option with land-sea mask considered 

44 extrapolation_mode (str): 

45 Mode to use for extrapolating data into regions beyond the limits 

46 of the input cube domain. Refer to online documentation for 

47 iris.analysis. 

48 Modes are - 

49 extrapolate - extrapolated points will take their values from the 

50 nearest source point 

51 nan - extrapolated points will be set to NaN 

52 error - a ValueError will be raised notifying an attempt to 

53 extrapolate 

54 mask - extrapolated points will always be masked, even if 

55 the source data is not a MaskedArray 

56 nanmask - if the source data is a MaskedArray extrapolated points 

57 will be masked; otherwise they will be set to NaN 

58 land_sea_mask_vicinity (float): 

59 Radius of vicinity to search for a coastline, in metres. 

60 regridded_title (str): 

61 New "title" attribute to be set if the field is being regridded 

62 (since "title" may contain grid information). If None, a default 

63 value is used. 

64 

65 Returns: 

66 iris.cube.Cube: 

67 Processed cube. 

68 

69 Raises: 

70 ValueError:
